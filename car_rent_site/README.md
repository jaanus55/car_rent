# Car Rent demo site

Valmis tehtud lihtne autorendi veebileht vastavalt ülesande README kirjeldusele.

## Funktsioonid
- PHP + mysqli
- Bootstrap 5 kujundus
- avaleht, autode list, detailvaade
- otsing `brand` ja `model` järgi
- pagination
- admin CRUD
- admin login sessiooniga
- kasutajad + reserveeringud
- kuupäevade kattuvuse kontroll

## Käivitamine
1. Loo MySQL andmebaas `car_rent`
2. Impordi `sql/schema.sql`
3. Vajadusel muuda `inc/config.php`
4. Käivita projekt näiteks: `php -S localhost:8000`
5. Ava `http://localhost:8000`
